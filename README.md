# antigravity_installation

本專案作為 **AntiGravity** 懶人包的安裝與測試專案環境。

## 目錄結構

- `01-antigravity-lazypack.md`：主安裝指引（整理 NotebookLM、Firebase、GitHub 與 Obsidian 的連接步驟）。
- `SKILL.md`：AI Agent 自動安裝入口說明。
- `AGENTS.md`：本專案的 AI Agent 工作規則與專案邊界。
- `100_Todo/`：任務與行動管理目錄。
- `200_Reference/`：參考素材與文件模板目錄。

## 專案使用方式

### 方式一：直接叫 AI 協助安裝設定（推薦）
把這行貼給您的 AI agent：
```text
這是 AntiGravity 懶人包 https://github.com/icestone0128/antigravity-installation
請讀取 repo 內容並依據 SKILL.md 引導我完成服務連接與工作流程設定。
```
AI 會自動讀取 `SKILL.md`，並引導您進行相關的環境檢查、服務連接與開收工設定。

### 方式二：手動開啟設定檔
1. 開啟 `01-antigravity-lazypack.md`。
2. 依照文件內之步驟，逐一完成環境檢查、OAuth 登入與各服務 MCP 設定。
